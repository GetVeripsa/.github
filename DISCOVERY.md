## Safe copy

Veripsa is pre-merge traffic control for AI coding agents. It surfaces GitHub checks and PR comments when open pull requests may need a safer landing order. It is advisory by default and is not a merge queue, AI code reviewer, CI replacement, or guarantee that every defect disappears.
